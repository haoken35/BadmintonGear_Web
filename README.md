## Hướng dẫn chạy dự án BadmintonGear_Web

Đây là hướng dẫn chi tiết để thiết lập và chạy dự án Front end Web của BadmintonGear.

## Các Bước Cài Đặt

### 1. Chuẩn Bị Biến Môi Trường

Tạo file .env và điền phù hợp vào các giá trị
NEXTNEXT_PUBLIC_SOCKET_URL = http://localhost:3000
NEXT_PUBLIC_API_URL=http://localhost:3000/api/v1

### 2. Cài Đặt Dependencies

```bash
npm install
```
### 3. Chạy Dự Án

Chạy server ở chế độ development:

```bash
npm run dev
```

Server sẽ khởi động tại `http://localhost:3030`
